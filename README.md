# Java CD Organizer

#### By _Ming Wen_

## Description

Remember CDs?

This is a program that allows users to catalog and organize their CDs. Functionality includes: create a new CD, enter the CD title, list out all CDs, add an artist, choose (for each CD) what artist the CD is by, list out all artists, and choose an artist to see which CDs are by that artist.

## Setup and Installation Requirements

  Clone the repo.  
  Run a instance of the gradle web server with 'gradle run'.  
  Navigate to localhost:4567

  Additionally, tests can be run with 'gradle test'.
